起床，乘车去菜市场，买菜，挑新鲜的，然后回来做法。中午收到一个从大陆寄来的快递，让我自己去邮局拿。然后我就骑自行车去拿回来，然后回来做饭。然后看一会儿电视 ( 小平板 ) ，傍晚出去散步运动休息 去银行去点儿钱 自由活动。晚上有去买了一个大披萨 用刷卡的 ( 有余额变动短信通知 ) ，晚上回来收到银行的电子账单，然后操作网银转账给还掉。
